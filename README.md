# Arena Survivor

A 2D top-down arena survival game built in Unity.

## Unity Version
Built with Unity 6000.3.9f1

## How to Open
1. Clone the repo
2. Open Unity Hub → Add → select the project folder
3. Open `Arena` scene in Assets/Scenes/
4. Hit Play

## What's in it so far
- 20x20 floor tilemap
- Wall tilemap around arena perimeter with Tilemap Collider 2D
- Camera framing the full arena
- Player GameObject with top-down sprite
- WASD movement using Rigidbody2D.MovePosition
- Player BoxCollider2D — wall collision working
